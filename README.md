# QTL - Premium AI Solver for 2048

QTL is a high-performance, visually stunning AI solver for the game 2048. It features a powerful Minimax algorithm with Alpha-Beta pruning, a premium glassmorphic interface, and manual placement support.

## ✨ Features

- **🤖 Intelligent Solver**: Powered by Minimax with heuristic evaluation optimized for high-tile strategy.
- **🎨 Premium Design**: Modern glassmorphism UI with vibrant gradients, fluid typography, and smooth CSS animations.
- **📱 Fully Responsive**: Optimized for Desktop (3-column layout), Tablet, and Mobile devices.
- **🏗️ Manual Mode**: Place tiles manually after any AI move to test specific scenarios or play along.
- **📊 Real-time Analysis**: View live evaluation scores for every possible move.

## 🚀 How to Run

QTL is a standalone web application. No installation is required.

1. Clone or download the repository.
2. Open `index.html` in any modern web browser.

## 🛠️ Built With

- **HTML5 & CSS3**: Vanilla implementation for maximum performance.
- **JavaScript**: Optimized AI logic without external dependencies.
- **Inter Font**: Modern typography for a premium look.
